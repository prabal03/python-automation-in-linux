Hey guys we have made this python-automation-in-linux which help you in following tools:

1) HADOOP WITH SHARING ELASTICITY STORAGE
In this the only dependency is you must have the soft of hadoop and jdk after downloading that software simply you can run this code which help you to form the cluster by just pressing the buttons.

2) INSTALLATION OF WEBSERVER
In this there is no dependency you simply press the key , your webservice will start .

3) AWS COMMANDS
In this there is no dependency , if you don not have the AWSCLI it will install for you and after installation done ,you can run various services given inside this.

4) BASIC LINUX COMMANDS 
Here also there is no dependency , inside this you can run basic commands of linux like creating directory , making of directory , deleting files inside the directory , checking connection between the systems and various commands .

5) LVM CONCEPT
In this also there is no dependency , inside this you can convert static disk to dynamic disk , cretaing physical volume , creating volume group , creating logical volume etc.

6) DOCKER SERVICES
In this also there is no dependency, you can run various commands releated to docker like , installing os , starting os , stoping os , deleting container etc. 


This automation you can run on your local systems and if you want to run in server-side you can also do this with the help of this automation.
The file name team.py is the main file which you gonna run and team1.py(server side file) is the file when you run program server side, this file will be sent on the server side when you give the server ip.

Put this files inside the root directory of your linux.
Now by this automation you dont have to remember the commands , you have to just press one button and your work will be done automatically.
